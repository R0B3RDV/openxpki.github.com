Roadmap / Releases

Release Cycle

We do not have an official release cycle but we plan to publish new releases whenever we add new functionality or in case we find any severe bugs.

Versioning

We follow the semantic versioning guidlines [[semver.org]], so you are able to update within the 1.x branch at any time without touching your config.

Roadmap / Version 1.x

  • REST API for request/renewal of certificates
  • Extend SOAP API for request/renewal of certificates
  • Dedicated renewal workflow
  • ....

Roadmap / Version 2.x

Those features are considered to require some more heavy lifting in the core and might break APIs, so they are scheduled for a 2.x release.
