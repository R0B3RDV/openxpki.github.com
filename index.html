The OpenXPKI Project

The OpenXPKI project has the vision to publish a software stack that provides all necessary components to manage keys and certificates based on the x509 cryptography standard.

Project Status

As of Summer 2015 the core and new webui are quite stable and we are doing some final polishing. The plan is to release one dot zero on the 10th anniversary of the project in October 2015. There are already several installations used in production and we encourage anybody to give it a try and provide feedback.

Core Features

  • WebUI compatible with all major browsers
  • Ready-to-run example config included
  • Support for SCEP (Simple Certificate Enrollment Protocol)
  • Easy adjustment of workflows to personal needs
  • Run multiple seperate CAs with a single installation, Automated rollover of CA generations
  • Can use Hardware Security Modules (e.g. nShield solo) for crypto operations
  • Issue certificates with public trusted CAs (e.g. SwissSign, Commodo, VeriSign)
  • Based on openssl and perl, runs on most *nix platforms
  • 100% Open Source, commercial support available
